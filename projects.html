Project Summary

Project Summary (Work In Progress)

  • A web application to store recipes and manage ingredients to make cooking easier
  • Keep track of what ingredients you have
  • Filter recipes by what you have available
  • Add recipes to a shopping list in one click

Technologies

  • Made use Next.js and React
  • Designed database structure and created with MySQL, set up on an Ubuntu Server machine
  • Implemented Rest API routes to handle database CRUD operations
  • Authentication implemented using JSON Web Tokens, and Bcrypt for password hashing
